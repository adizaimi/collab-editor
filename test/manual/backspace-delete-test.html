Instructions

  1. Click "Setup" button to prepare the test with "HELLO" text

  3. For Backspace test: Press Backspace key - should delete first L → "HELO"

Key Difference:

  • Backspace: Deletes character BEFORE cursor (to the left)
  • Delete: Deletes character AFTER cursor (to the right/at cursor)
